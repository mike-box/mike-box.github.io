5181. 公交站间的距离

题目

环形公交路线上有 n 个站，按次序从 0 到 n - 1 进行编号。我们已知每一对相邻公交站之间的距离，distance[i] 表示编号为 i 的车站和编号为 (i + 1) % n 的车站之间的距离。

环线上的公交车都可以按顺时针和逆时针的方向行驶。

返回乘客从出发点 start 到目的地 destination 之间的最短距离。

https://leetcode-cn.com/contest/weekly-contest-153/problems/distance-between-bus-stops/

题意

求两个公交车站之间的最短距离。

思路

  1. 题目本身比较简单，求数组中两个索引之间的距离。
  2. 由于本身数组为循环数组，所以只需要求出两个方向的距离即可，顺时针和逆时针。
  3. dist(i,j) = min(sum(i,j),sum(0,n)-sum(i,j)),非常简单的求和计算即可完成。

5182. 删除一次得到子数组最大和

题目

给你一个整数数组，返回它的某个 非空 子数组（连续元素）在执行一次可选的删除操作后，所能得到的最大元素总和。

换句话说，你可以从原数组中选出一个子数组，并可以决定要不要从中删除一个元素（只能删一次哦），（删除后）子数组中至少应当有一个元素，然后该子数组（剩下）的元素总和是所有子数组之中最大的。

注意，删除一个元素后，子数组 不能为空。

地址

https://leetcode-cn.com/contest/weekly-contest-153/problems/maximum-subarray-sum-with-one-deletion/

题意

LCS问题，求出该数组的最大子序列和，或者删除一个元素后的最大子序列和。

思路

  1. 子数组和的最大值可以用典型的lcs。

  2. 求删除一个元素后的子数组的最大和，可以用递推公式$ans = max(ans,left[i-1] + right[i+1])$

    class Solution:
    def maximumSum(self, arr: List[int]) -> int:

     n = len(arr)
 left = [arr[0] for _ in range(n)]
 right = [arr[n-1] for _ in range(n)]
 ans = max(left[0],right[n-1])

 for i in range(1,n):
     left[i] = arr[i] + max(0,left[i-1])
     right[n-1-i] = arr[n-1-i] + max(0,right[n-i])
     ans = max(ans,left[i])
     ans = max(ans,right[n-1-i])

 for i in range(1,n-1):
     ans = max(ans,left[i-1]+right[i+1])

 return ans

    输入：arr1 = [1,5,3,6,7], arr2 = [1,3,2,4]
    输出：1
    解释：用 2 来替换 5，之后 arr1 = [1, 2, 3, 6, 7]。

    输入：arr1 = [1,5,3,6,7], arr2 = [4,3,1]
    输出：2
    解释：用 3 来替换 5，然后用 4 来替换 3，得到 arr1 = [1, 3, 4, 6, 7]。

    输入：arr1 = [1,5,3,6,7], arr2 = [1,6,3,3]
    输出：-1
    解释：无法使 arr1 严格递增。

    class Solution {
    public:
    int makeArrayIncreasing(vector& arr1, vector& arr2) {

     int n = arr1.size();
 std::vector<vector<int>> dp(n+1,vector<int>(n+1,INT_MAX));

 /*intial*/
 dp[0][0] = -1;
 sort(arr2.begin(),arr2.end());

 for(int i = 1; i <= n; ++i){
     for(int j = 0;j <= i; ++j){
         if(arr1[i-1] > dp[j][i-1]){
             dp[j][i] = arr1[i-1];
         }

         if(j > 0){
             auto it = upper_bound(arr2.begin(),arr2.end(),dp[j-1][i-1]);
             if(it != arr2.end()){
                 dp[j][i] = min(dp[j][i],*it);
             }
         }
         if( i == n && dp[j][i] != INT_MAX){
             return j;
         }
     }
 }

 return -1;

    }
    };
