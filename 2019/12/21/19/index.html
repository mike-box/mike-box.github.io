一道高中的数学题目

题目

  • 从 （0， 1）中随机选择两个数，两个数的积小于 1/3的概率是_

题意

  • 假设有两个随机数 x,y,满足条件（x > 0 && x < 1， y > 0 && y < 1),求x*y < 1/3的概率，即求p(xy < 1/3).
  • 主要是求概率

解题思路

  1. 求概率其实是求在约束条件下，几个函数所围成的图形的阴影面积.
  2. 直线x = 0, x= 1, y = 0, y = 1, xy = 1/3,所围成的阴影面积.
  1. 本题的解法如下所示，利用递推公式，积分即可得到阴影部分的面积.
  2. 最终结果为1/3(1+ln3).
